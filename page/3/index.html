哥斯拉大战金刚2：帝国崛起

发表于 2024-03-30 更新于 2025-09-02 分类于 后感 阅读次数：

周末懒得出门，很久没去过电影院了。今天出去吃了个饭刚好附近有个影院又有电影要放就顺便去看了一下。

3D效果还是挺震撼的，就是戴两份眼镜的感受不算太好，也许下次可以试试自己买个3d眼镜夹片？

电影没太多墨迹的文戏，怪兽的打戏时长还是很足的，就是稍微有点隔靴搔痒，虽然从头打到尾，但双方都像是给彼此挠痒痒，就算被打飞了下一秒也满血复活。全篇唯一的两次受伤，一次是金刚被冻伤，但也是被动力手套注射一剂马上康复，一次是刀疤最后被冻死。说到刀疤，感觉这个选角并不太适合当大反派，细长的身形没什么气势，也没什么压迫感，更适合当反派军师或者打手之类的角色，以至于一开始雪魔那两个眼睛现身的时候，我还以为它才是那个反派猩猩王。

全片给个七八分吧，没什么太精彩的镜头，但打发个时间还行。

最后，从影院出来的时候没看地图，走了十分钟反方向的路。。

墨水屏

本科的时候买过一次kindle，现在记忆也不太清楚了，但印象里还行，最后是因为没什么想看的书又出掉了。

最近上班空闲时间比较，看些杂七杂八的小说，又有点想买墨水屏了，但总感觉现有的品类都不太满足需求，理想的形态大概是7寸+黑白屏+人体工学握持+开放系统吧，续航需求应该倒是不太有，但各家产品总有一两个条件满足不了。

然后最后想了想，又感觉其实没太多买的必要，想买其实主要是因为手机的反光比较强，总要挑下角度不然灯管反射出来根本看不了。但后来看了下视频，墨水屏其实也有反射，最多缓和一点但不可能跟书那样，所以现在大概也没那么想换了。

波斯王子：失落王冠

发表于 2024-03-30 更新于 2025-09-02 分类于 后感 阅读次数：

很久没感到这种属于游戏的简单快乐了。前几天刚玩圣兽之王的时候也还行，但后续感觉肉眼可见的重复，而失落之冠我觉得自己可以通关。

我的理想游戏大概就是这种十几二十小时的动作游戏吧，横版还是3d倒无所谓，包括之前的暗邪西部，虽然别人的评价不算高，我玩起来还是挺高兴的。

但还是有一点比较糟心，育碧果然会有bug，和瓦赫拉姆对决后莫名其妙进入了无敌状态，不仅怪打不到我，连陷阱都无法造成伤害，一开始还没意识到，直到穿过了好几个转轮和毒潭一路飞速向下后才发现不对劲，这时候已经连影子技能都拿到了。而且这bug退到主菜单再读档都没用，必须退出游戏才行。总之下水道这部分地图废了一大半，都不是按照正常路径探索的，但考虑到这部分地图有点恶心，而且前半段还不给大地图，跳过就跳过了吧。希望之后别再出现类似的bug。之前看一些视频的评论，甚至有人遇到坏档bug，要是碰到了大概也只能弃游了，不可能有耐心再打个几小时追进度。

通关了，整体体验还行，但后期战斗少点味，小怪种类不太多，对战也基本是老一套搞定。BOSS的话有点缺少互动，就是躲躲躲然后砍几刀，也没几招能弹反。跳跳乐大部分还行，但有几个过于恶心，尤其是那种上下尖刺还在位移的圆柱，一不小心就会碰到刺，而且有的还要求超长时间零失误才行，有点折磨，过了也没太多收益和成就感。总之通关后没太多补全图的兴趣。

我不是戏神

看不出作者想塑造一个怎么样的主角，既想塑造成对抗恶势力为全人类奋斗的正义形象调动情绪，又自然而然地为了进阶杀光队友。虽然事后补了一句有没主角都要死，但在主角亲自动手杀了几个的情况下这种找补毫无力度。

新手教程给人的感觉有点过于糟糕，在没有路线指引只有目标点提示的情况下，极其频繁地将目标设定在墙后/背后，让人完全不知道该往哪走，再加上受限的体力，让人在迷茫的时候还毫无探索的欲望。以及那个跳到灯柱上射墙后信标的设计，配上空气墙，可以说是设计得非常荒诞和反直觉。
