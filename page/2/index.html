墨西哥钝口螈

墨西哥钝口螈出现一个特性叫幼态延续，意思是它们由幼体发育到性成熟期间，没有经历变态。不经历变态原因是它们缺乏甲状腺荷尔蒙。

海兽

海牛，可能由长鼻目（象）进化而来。

脑

发表于 2022-01-11 更新于 2022-01-12 分类于 杂谈 阅读次数：


在《祖先的故事》中了解到，通常脑容量是与体重正相关的，因此大型动物可以有着一个比人类大得多的大脑。这看起来是很自然的，因为越大的体型意味着有越多的肌肉需要控制，但除去肌肉的控制之外呢？如果是要计算1+1，或者区分红与蓝，不同体型的动物需要的神经元会有区别吗？《祖先的故事》中认为，双对数轴可以更直观地反应一个物种大脑比应有的大上多少倍，但大脑能容纳的智慧是否真的应该只用倍数、而非一个绝对的差值来比较呢？对鲸鱼这样的巨型物种，一次10%的脑容量增加是否就足以为智慧的孕育提供潜在的可能呢？

话语

发表于 2021-12-17 分类于 杂谈 阅读次数：

语言是交流的工具，是信息的传递者。但除却纯粹的信息交换，比如现在是几点、火车几点出发，剩下的一切话语都天然地被发言人渲染上自身的色彩。这一点上几乎不存在中立者，属于中立者的身份通常是沉默。一切难以进行彻底量化的事物（世上多数事物都是如此）在表达时都将经历语言的压缩、取舍，被冠上多样的名号，却永远无法从这些名号还原其本身。人们所能做的只是抢占各自的山头，各说各话。
