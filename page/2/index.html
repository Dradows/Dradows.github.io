双城之战2

发表于 2024-11-24 更新于 2025-09-02 分类于 后感 阅读次数：

我很讨厌任何作品里的平行宇宙的设定，除非这就是作品的主题本身（即使是这类也少有能诠释好这个设定的）。也许编剧觉得一万个不同的可能性同时发生会很酷， 但在我看来这纯粹是消费观众的情感。当一切都留有余地，一切都还有可能，那现在的行为又有什么意义呢？这种背景下无法产生真正厚重的情绪。 而当主题从个体转向了某种更宏大且更神秘的奇幻设定后，也更难引起观众的共鸣。

常用正则表达式

发表于 2024-04-15 更新于 2025-09-02 分类于 技术 阅读次数：

参考 正则表达式 - JavaScript | MDN (mozilla.org)

字符 含义
^ 匹配输入的开始。如果多行标志被设置为 true，那么也匹配换行符后紧跟的位置。
$ 匹配输入的结束。如果多行标志被设置为 true，那么也匹配换行符前的位置。
* 匹配前一个表达式 0 次或多次。等价于 {0,}。
+ 匹配前面一个表达式 1 次或者多次。等价于 {1,}。
? 匹配前面一个表达式 0 次或者 1 次。等价于 {0,1}。
. （小数点）默认匹配除换行符之外的任何单个字符。
(x) 它会匹配 'x' 并且记住匹配项。其中括号被称为捕获括号。
(?:x) 匹配 'x' 但是不记住匹配项。这种括号叫作非捕获括号 。
x\|y 匹配‘x’或者‘y’。
{n} n 是一个正整数，匹配了前面一个字符刚好出现了 n 次。
比如， /a{2}/ 不会匹配“candy”中的'a',但是会匹配“caandy”中所有的 a，以及“caaandy”中的前两个'a'。
{n,} n 是一个正整数，匹配前一个字符至少出现了 n 次。例如，/a{2,}/ 匹配 "aa", "aaaa" 和 "aaaaa" 但是不匹配 "a"。
{n,m} n 和 m 都是整数。匹配前面的字符至少 n 次，最多 m 次。
[xyz] 一个字符集合。匹配方括号中的任意字符，包括转义序列。
[^xyz] 一个反向字符集。也就是说， 它匹配任何没有包含在方括号中的字符。
\b 匹配一个词的边界。
\d 匹配一个数字。等价于 [0-9]。
\s 匹配一个空白字符，包括空格、制表符、换页符和换行符。
\w 匹配一个单字字符（字母、数字或者下划线）。等价于 [A-Za-z0-9_]。例如，/\w/ 匹配 "apple," 中的 'a'，"$5.28,"中的 '5' 和 "3D." 中的 '3'。
\1 在正则表达式中，它返回最后的第 n 个子捕获匹配的子字符串。
方法 描述
exec 一个在字符串中执行查找匹配的 RegExp 方法，它返回一个数组（未匹配到则返回 null）。
test 一个在字符串中测试是否匹配的 RegExp 方法，它返回 true 或 false。
match 一个在字符串中执行查找匹配的 String 方法，它返回一个数组，在未匹配到时会返回 null。
matchAll 一个在字符串中执行查找所有匹配的 String 方法，它返回一个迭代器（iterator）。
search 一个在字符串中测试匹配的 String 方法，它返回匹配到的位置索引，或者在失败时返回 -1。
replace 一个在字符串中执行查找匹配的 String 方法，并且使用替换字符串替换掉匹配到的子字符串。
split 一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组中的 String 方法。

玩完匹诺曹后想起了艾尔登法环一直没通关，今天重新体验了几分钟，已经有点接受不了了。首先双持的话没防御，只能闪避，闪避又很笨重，战斗方面的体验并不算很好，而且开荒的时候就是猎犬长牙，今天进游戏看到双持大曲，有种熟悉又无聊的感觉。

公司edge配置

发表于 2024-04-07 更新于 2025-09-02 分类于 技术 阅读次数：

公司电脑的edge有几个问题

  1. 关闭浏览器后密码自动清除

  2. 点击下载后延迟弹出下载栏

  3. ctrl + c复制偶尔会不生效。

之前是换火狐试了一段时间，这几个问题倒是都解决了，但有个网站必须用chromium内核才能访问，还是准备换回edge，不然浏览器来回切有点麻烦。

保存密码

删除密码和浏览器内的设置无关，应该是通关删除用户文件实现的，所以换个路径就好了。

捅过注册表，在HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft添加Edge项，在Edge项添加字符串参数UserDataDir为另一路径D:\Edge\即可。

取消下载延迟

直接在设置中搜索Microsoft Defender Smartscreen取消即可。

复制失效

升了个新版本，目前好像没出现过了，要是碰到了再说。

匹诺曹的谎言

发表于 2024-04-07 更新于 2025-09-02 分类于 后感 阅读次数：

上次买波斯王子的赶上优惠，200元买了波斯王子+渡神纪+看门狗军团+刺客信条幻景，但玩完波斯王子后，一想到之前玩刺客信条和孤岛惊魂的经历，突然就没玩另外三个游戏的兴致了。渡神纪倒是玩了一小会，也没太有继续玩的欲望。

然后过了几天不知道玩什么的日子，在steam和xgp来回翻了一下还有哪些游戏，突然看到匹诺曹，于是试了一下，比想象中好玩，甚至比黑魂还要合我口味。黑魂不记得为啥，玩几次都没玩多久就又退坑了。

匹诺曹的地图相对线性，分支基本都是那种放一两个蓝点或者宝箱的短分支以及往回连接的捷径分支，探索起来还是比较有意思的，也不会有什么选择困难的纠结情况，基本很容易看出来哪边是主线哪边是分支。不过缺了大地图，有些支线任务又要回老地图，但拿到道具后早忘了原先的路了，但现翻下攻略也还能接受。
