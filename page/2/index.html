寻枪

发表于 2021-08-24 分类于 后感 阅读次数： Valine：

是被B站的推送标语吸引去看的，类似于“丢了枪后初恋女友被杀死”，具体的忘了。看上去有些悬疑意味，实际上十分朦胧。网上有许多“深入解读”，但看上去都是些似是而非的。有人说马军精神分裂，举出一堆像是“最后镜头只有一个人”，“周小刚说‘你们都死了’”之类的例子，如果逐帧仔细看明显可以看到警察包围前地上有一蹲一躺的两个人，只是后来包围住了不明显，而周小刚说的话显然也只是断句问题，如果真是幻想中的人周小刚又怎么可能和他们对话，更不用说一开始就是老树精送他们回的家。

在我看来整个故事只是个简单的丢枪，找枪，用命留枪的故事，在这之上又添加了一些特殊的艺术化的表达手法，最多是床上说“假枪”这种黄色的双关与心理表达，但并没有对另一种剧情的暗示。许多人也许更倾向于一种神秘的、不可证实的解读方式吧。

发表于 2021-08-24 分类于 闲话 阅读次数： Valine：

铭印（英语：imprinting），又称印记、印迹、印随或印痕。在行为生物学中指的是一种不可逆的学习模式：通常在一段比较短的，由基因决定的时期里（敏感时期），环境的刺激会被长久地植入个体的行为中，后来看来就好像先天习得的一样。本能理论认为，铭印是通过学习，而掌握了钥匙刺激而形成的现象。
后代铭印（Filial imprinting）让幼体学会辨认母亲。某些生物出生以后会紧跟着它第一眼见到的较大的可移动的物体而走。印痕作用一旦发生，就很难改变。由于生物出生后，第一眼看到的物体通常是其母亲，所以生物的这种行为可以使得生物幼体出生后不易认错它们的母亲，从而得到母亲的保护。如果生物幼体第一眼看到的不是它的母亲，它可以跟随任何较大的物体移动，甚至是非生命的物体。

人在懵懂时期遇到的第一个异类，或许也会被奉为圭臬。
